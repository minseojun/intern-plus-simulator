# 인턴+ (Intern+)
**실전 업무 시뮬레이터 — 마케팅 직무**

감성 라이프스타일 앱 스타트업 **루나틱 코퍼레이션**의 마케팅 담당자로 출근해, 실제 업무 상황을 체험하는 시뮬레이션 게임입니다. 이메일 대응, 보고서 작성, 위기 의사결정을 처리하면 AI가 행동 전체를 종합 평가합니다.

---

## 실행 방법

### AI 기능 포함 (권장)
1. 이 레포지토리를 ZIP으로 다운로드하거나 `git clone`
2. 폴더 안에 `config.js` 파일 생성 (아래 참고)
3. `index.html`을 브라우저로 열기

### config.js 만드는 법
프로젝트 폴더 안에 `config.js` 파일을 직접 만들고 아래 내용을 붙여넣은 뒤, API 키를 입력하세요.

```javascript
const CONFIG = {
  API_KEY: 'sk-ant-여기에-본인-API-키-입력',
  MODEL: 'claude-haiku-4-5-20251001',
};
```

> ⚠️ API 키는 [Anthropic Console](https://console.anthropic.com)에서 발급받을 수 있습니다.  
> ⚠️ `config.js`는 절대 GitHub에 올리지 마세요. `.gitignore`에 이미 포함되어 있습니다.

### AI 기능 없이 실행
`config.js` 없이 `index.html`을 열어도 게임은 실행됩니다. 단, 라운드 평가와 최종 리포트 기능은 작동하지 않습니다.

---

## 게임 방법

| 키 | 동작 |
|---|---|
| `W A S D` | 오피스 이동 |
| `E` | NPC에게 말 걸기 / 자리에 앉기 |
| `ESC` | 대화 닫기 |

1. WASD로 오피스를 돌아다니며 동료 NPC에게 말을 걸 수 있습니다
2. 파란 데스크(내 자리)에 가까이 가서 `E`를 누르면 업무 화면으로 전환됩니다
3. 이메일에 답장하고, 보고서를 작성하고, 업무 칸반 보드를 관리합니다
4. 라운드당 **7분** 안에 필수 과제를 완료하세요
5. 라운드 종료 후 AI가 행동 전체를 종합 평가합니다

## 라운드 구성

| 라운드 | 요일 | 난이도 | 핵심 상황 |
|---|---|---|---|
| R1 | MON | 기본 | 성과 보고서 마감, 광고 소재 승인, SNS 전략 피드백 |
| R2 | TUE | 중급 | 서버 과부하 위기, 추가 예산 집행, 경쟁사 공세 |
| R3 | WED | 고급 | SNS 위기 대응, 인플루언서 계약 위반, 예산 삭감 |

## AI 평가 항목

라운드 종료 시 아래 4가지를 종합적으로 평가합니다.

- **대응력** — 이메일 열람·답장의 적시성과 완성도
- **보고서 품질** — 데이터 인용, 원인 분석, 전략 제안의 구체성
- **의사결정** — 판단의 명확성과 근거
- **시간 관리** — 우선순위 파악과 처리 효율

> 보고서 평가는 엄격합니다. 데이터 없이 두루뭉술하게 쓰면 낮은 점수를 받습니다.

3라운드 완료 후 종합 피드백 리포트 (업무 유형, 강점/약점, 추천 직무, 액션 플랜) 를 제공합니다.

```

├── .gitignore       config.js 제외 설정

```

## 주의사항

- `config.js`는 GitHub에 올리지 마세요. API 키가 공개됩니다.
- AI 기능은 로컬 실행(`index.html` 직접 열기) 환경에서만 작동합니다.
- GitHub Pages로 배포하면 게임 UI는 보이지만 AI 평가 기능은 작동하지 않습니다.
