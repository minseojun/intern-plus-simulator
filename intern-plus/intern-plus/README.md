# 인턴+ (Intern+) Office Simulator

실제 업무환경을 시뮬레이션해 지원자의 실무 수행 능력과 행동 패턴을 AI로 분석하는 채용·온보딩 평가 플랫폼 프로토타입입니다.

## 파일 구조

```
intern-plus/
├── index.html      — HTML 구조 (게임 화면, 데스크 UI, 오버레이)
├── style.css       — 전체 스타일 (라이트 테마, 오피스 UI)
├── rounds.js       — 3개 라운드 데이터 (이메일, 과제, NPC 대화)
├── game.js         — 캔버스 렌더링, 게임 로직, AI 코치 패널
├── evaluation.js   — 라운드 평가, 최종 피드백 리포트
└── README.md
```

## 실행 방법

1. 파일을 모두 같은 폴더에 두고 `index.html`을 브라우저에서 열기
2. 또는 GitHub Pages로 배포 (Settings → Pages → Deploy from main branch)

> ⚠️ Claude API를 직접 호출하므로 `claude.ai` 아티팩트 환경에서만 동작합니다.  
> 로컬 또는 자체 서버에서 사용하려면 API 키 인증 로직을 추가해야 합니다.

## 게임 방법

- **WASD** — 오피스 이동
- **E** — NPC 대화 / 자리 앉기
- 자리에 앉으면 모니터 화면 전환 (이메일 · 보고서 · 업무 탭)
- 라운드당 **7분** 안에 필수 과제 완료

## 라운드 구성

| 라운드 | 난이도 | 핵심 과제 |
|---|---|---|
| R1 | 기본 | 성과 보고서 작성, 광고 소재 승인, SNS 캘린더 피드백 |
| R2 | 중급 | 서버 위기 의사결정, 예산 집행 판단, 경쟁사 대응 |
| R3 | 고급 | SNS 위기관리, 인플루언서 계약 위반, 예산 삭감 재조정 |

## AI 평가 항목

- 대응력 (이메일 열람 · 답장 여부)
- 작성 품질 (보고서 내용 · 이메일 답장 수준)
- 우선순위 판단
- 처리 속도

3라운드 완료 후 종합 피드백 리포트 (업무 유형, 강점/약점, 추천 직무, 다음 단계) 제공
